SAE Reporting Project

Context

As part of the development of an automated system for managing data from different CAFs in French regions, several CSV files were provided. To accomplish this mission, we used various tools and software, including SAS, SQL, Excel, and Power BI.

Constraints

  • Insufficient skills for designing the automated tool in SAS.
  • Difficulty in reading and understanding official documentation.

Steps

The steps followed a logical plan:

In SAS

  • Importing data
  • Creating automated processes (SQL Macros and SymputX)
  • Processing data
  • Formatting
  • Exporting results to Excel format

In Excel

  • Data layout
  • Creating smart adjustable tables
  • Exporting results to PDF

In Power BI

  • Importing Excel file
  • Creating relational models
  • Creating an interactive dashboard

SAE SID Project (Decision Support Information Systems)

Context

As part of a university project, we studied video game sales on various online platforms. Using three datasets from Kaggle, we utilized Power BI for data analysis and visualization, Microsoft Word for the final report, and PowerPoint for the presentation. This project enhanced our skills in data management, decision-making analysis, and communication.

Constraints

  • Searching for useful and sufficiently large datasets.
  • Difficulty in reading and understanding official documentation.

Approach

The steps followed a logical plan:

Data Research

  • Learning about the subject
  • Searching for datasets on Kaggle
  • Verifying the datasets
  • Importing files in CSV format

Power BI

  • Importing CSV files
  • Cleaning the datasets
  • Correcting missing values and creating new variables
  • Creating star schema relational models
  • Designing interactive dashboards

Writing the Final Report

  • Structuring the documents
  • Explaining the initial process
  • Inserting visuals and final formatting
  • Exporting the document in PDF format
